# Knative

Knotters native application using react-native webview.

## Setup project

Create a .env file in project's root using reference from the existing [.env.example](.env.example) file.

## Install dependencies

```bash
yarn
```

## Run application

```bash
yarn start
```

```bash
yarn android
```

```bash
yarn ios
```

## Testing

```bash
yarn test
```
